> Receiving messages based on a pattern (topics)

## Install

```sh
yarn install
```

## Usage

Open 1st terminal with below command

```sh
yarn receive "info.*"
```

Then open other terminal to emit message to rabbit mq

```sh
yarn emit info.1 hello
yarn emit info.2 demo
```

## Run tests

```sh
yarn test
```
